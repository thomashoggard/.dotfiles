# Neovim Quick Reference

## Prerequisites
- live-server
- pandoc
- fzf
- fzy
- ripgrep

Install Language Servers:

```
npm install -g typescript-language-server typescript
```

Install daemons
```
npm install -g prettier_d_slim
npm install -g eslint_d
```

## Commands

Preview a markdown file -
```
:MarkdownPreview
```
